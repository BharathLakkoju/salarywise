# SalaryWise 💰

SalaryWise is a web application that helps users explore, compare, and understand salary trends across different roles, companies, and locations.

## 🚀 Features
- 📊 Compare salaries across roles and companies
- 🌍 Location-based salary insights
- 📈 Clean UI for data visualization
- ⚡ Built with performance-focused Next.js

## 🛠 Tech Stack
- Next.js
- React
- Tailwind CSS

## 📦 Installation
```bash
git clone https://github.com/BharathLakkoju/salarywise
cd salarywise
npm install
npm run dev
```

## 🎯 Use Case
Helps job seekers make better salary decisions during job switches.
